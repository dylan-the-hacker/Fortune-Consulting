# Fortune Consulting — Software Solutions

> A clean, production-grade landing page for Fortune Consulting, a Cape Town-based software solutions consultancy helping small businesses and startups find the right technology.

---

## Live Site

**[fortuneconsulting.netlify.app](https://fortuneconsulting.netlify.app)**

---

## Overview

Fortune Consulting bridges the gap between small businesses and the technology they need to grow. This site serves as the primary web presence — designed to communicate trust, clarity, and professionalism without any jargon.

---

## Features

- Fully responsive — mobile, tablet and desktop
- CSS-only animations with staggered entrance reveals
- Fixed navigation with blur backdrop
- Hero section with dot-grid background and stats bar
- Services strip
- About section with consultant card and credential grid
- Contact call-to-action band
- Zero dependencies — pure HTML, CSS, and JS

---

## Tech Stack

| Layer | Choice |
|---|---|
| Markup | HTML5 |
| Styling | Vanilla CSS (custom properties) |
| Scripts | Vanilla JS |
| Fonts | Playfair Display + Jost (Google Fonts) |
| Hosting | Netlify |


Or just drag `index.html` into your browser.
